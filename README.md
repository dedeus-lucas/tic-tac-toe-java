# Tic Tac Toe - Java

## 🇧🇷 Versão em Português

Projeto desenvolvido como parte da formação da Ada Tech, na conclusão do módulo de Lógica de Programação I.

Implementação de um jogo da velha em Java com foco em lógica, controle de fluxo e boas práticas de desenvolvimento.

---

### 🚀 Funcionalidades

- Jogo para dois jogadores (X e O)
- Entrada de jogadas via terminal
- Validação de entradas inválidas
- Verificação de vitória (linhas, colunas e diagonais)
- Detecção de empate
- Alternância automática de jogadores

---

### 🧠 Conceitos aplicados

- Arrays e matrizes
- Estruturas de repetição (while, for)
- Condicionais (if/else)
- Separação de responsabilidades (Board, Game, Main)
- Validação de entrada (Scanner)

---

### 🏗️ Estrutura do projeto
```
src/
├── Main.java    // execução do jogo
├── Game.java    // controle de jogadores
└── Board.java   // estado do tabuleiro
```
---

### ▶️ Como executar

1. Compile o projeto:

javac Main.java Game.java Board.java

2. Execute:

java Main

---

### 🎮 Como jogar

- O jogo solicita linha e coluna (valores de 1 a 3)
- Os jogadores se alternam automaticamente
- O jogo termina quando:
    - Um jogador vence
    - Ocorre empate

---

### 📌 Observações

- Projeto desenvolvido com Java puro (JDK 21)
- Sem uso de frameworks
- Foco em aprendizado e prática de lógica

---

Projeto desenvolvido para fins educacionais.

## 🇺🇸 English Version

Project developed as part of the Ada Tech program, as the final assignment of the Programming Logic I module.

Implementation of a Tic Tac Toe game in Java, focusing on logic, control flow, and clean development practices.

---

### 🚀 Features

- Two-player game (X and O)
- Console-based input
- Input validation for invalid entries
- Win detection (rows, columns, diagonals)
- Draw detection
- Automatic player switching

---

### 🧠 Concepts Applied

- Arrays and matrices
- Loop structures (while, for)
- Conditionals (if/else)
- Separation of concerns (Board, Game, Main)
- Input validation (Scanner)

---

### 🏗️ Project Structure
```
src/
├── Main.java    // game execution
├── Game.java    // player control
└── Board.java   // board state
```
---

### ▶️ How to Run

1. Compile the project:

javac Main.java Game.java Board.java

2. Run:

java Main

---

### 🎮 How to Play

- The game asks for row and column (values from 1 to 3)
- Players alternate automatically
- The game ends when:
    - A player wins
    - A draw occurs

---

### 📌 Notes

- Built using pure Java (JDK 21)
- No frameworks used
- Focused on learning and practicing programming logic
